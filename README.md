## Backend API
All form submissions and AI endpoints route through the TerrellOS backend on Fly.io:

**Base URL:** `https://terrellos-backend.fly.dev/v1/legion/`

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/health` | GET | Post info & status |
| `/info` | GET | Programs list |
| `/events` | GET | Steak Night, Fish Fry, Banquet |
| `/contact` | POST | Contact form |
| `/membership/apply` | POST | New member application |
| `/membership/renew` | POST | Renewal |
| `/volunteer` | POST | Volunteer signup |
| `/hall-rental` | POST | Hall rental request |
| `/events/register` | POST | Ticket registration |
| `/donate` | POST | Donations |
| `/military-ai/query` | POST | Military Guide AI |
